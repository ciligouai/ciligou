基础使用教程

界面介绍

磁力狗采用直观的界面设计，主要分为以下几个区域：

  • 顶部导航栏：包含主要功能入口和用户设置
  • 搜索区域：核心搜索功能，支持多种搜索方式
  • 结果展示区：显示搜索结果和相关信息
  • 侧边栏：提供快捷功能和历史记录
  • 底部状态栏：显示系统状态和连接信息

搜索功能使用

基本搜索

  1. 在搜索框中输入关键词
  2. 选择搜索类型（全部、文件、目录等）
  3. 点击搜索按钮或按回车键
  4. 查看搜索结果列表

高级搜索

  • 文件大小：指定文件大小范围
  • 文件类型：按文件扩展名筛选
  • 上传时间：按时间范围筛选
  • 种子数量：按种子数量筛选

下载管理

添加下载任务

  1. 在搜索结果中点击下载按钮
  2. 选择保存位置和文件名
  3. 设置下载优先级和限速
  4. 确认开始下载

管理下载任务

  • 暂停/恢复：控制下载进度
  • 删除任务：移除不需要的下载
  • 限速设置：控制下载速度
  • 队列管理：调整下载顺序

常见问题解决

🔍 搜索无结果

可能原因：关键词拼写错误、网络连接问题、服务器维护

解决方法：

  1. 检查关键词拼写
  2. 尝试使用同义词
  3. 检查网络连接
  4. 稍后重试

⬇️ 下载速度慢

可能原因：网络带宽限制、种子数量少、服务器负载高

解决方法：

  1. 检查网络带宽
  2. 选择种子数量多的资源
  3. 调整下载设置
  4. 使用多个下载源

💾 存储空间不足

可能原因：磁盘空间不足、临时文件过多

解决方法：

  1. 清理磁盘空间
  2. 删除不需要的下载
  3. 清理临时文件
  4. 更改下载目录

🔐 账号登录失败

可能原因：密码错误、账号被锁定、网络问题

解决方法：

  1. 检查密码是否正确
  2. 重置密码
  3. 检查网络连接
  4. 联系客服支持
