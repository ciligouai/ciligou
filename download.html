安装指南

Windows 安装步骤

  1. 下载 .exe 安装包
  2. 双击运行安装程序
  3. 按照安装向导提示操作
  4. 选择安装目录和组件
  5. 等待安装完成
  6. 启动磁力狗应用程序
注意：如果遇到权限问题，请右键选择"以管理员身份运行"

macOS 安装步骤

  1. 下载 .dmg 安装包
  2. 双击打开 .dmg 文件
  3. 将磁力狗拖拽到应用程序文件夹
  4. 从启动台或应用程序文件夹启动
  5. 首次运行可能需要在系统偏好设置中允许
注意：如果提示"无法验证开发者"，请在系统偏好设置 → 安全性与隐私中允许运行

Linux 安装步骤

Ubuntu/Debian 系统：

sudo dpkg -i ciligou_2.1.0_amd64.deb
sudo apt-get install -f

CentOS/RHEL 系统：

sudo rpm -i ciligou-2.1.0-1.x86_64.rpm

源码编译安装：

tar -xzf ciligou-2.1.0.tar.gz
cd ciligou-2.1.0
./configure
make
sudo make install

Android 安装步骤

  1. 下载 .apk 安装包
  2. 在设置中允许"未知来源"应用安装
  3. 点击下载的 .apk 文件
  4. 按照安装提示完成安装
  5. 从应用列表启动磁力狗
注意：建议从官方渠道下载，确保安全性

iOS 安装步骤

  1. 打开 App Store
  2. 搜索"磁力狗"
  3. 点击"获取"按钮
  4. 输入 Apple ID 密码或使用 Touch ID/Face ID
  5. 等待下载和安装完成
  6. 从主屏幕启动应用
注意：需要有效的 Apple ID 和网络连接
